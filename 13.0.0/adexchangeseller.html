Namespace: adexchangeseller

adexchangeseller v1

Ad Exchange Seller API

Accesses the inventory of Ad Exchange seller users and generates reports.

Version:
  • v1

Example

var google = require('googleapis');
var adexchangeseller = google.adexchangeseller('v1');
